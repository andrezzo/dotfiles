##### Clone repo at ~/:
<ul>
    <li>cd ~/</li>
    <li>git clone https://github.com/andrezzo/dotfiles.git</li>
</ul>

##### Remove dotfiles:
<ul>
    <li>cd ~/</li>
    <li>rm .vim</li>
    <li>rm .vimrc</li>
</ul>

##### Create links at ~/:
<ul>
    <li>ln -s .dotfiles/.vim .vim</li>
    <li>ln -s .dotfiles/.vimrc .vimrc</li>
</ul>
